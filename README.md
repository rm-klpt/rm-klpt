# Algorithms for superspecial abelian surfaces with maximal real multiplication

The repository contains code accompanying an anonymous submission.

## Structure
- The folder `rm_cgl` contains a proof-of-concept implementation of (2,2)-isogeny graph traversal for principally polarised superspecial abelian surfaces, with tracking of a real multiplication action on the 2^r-torsion.
- The folder `rm_klpt` contains an implementation of a generalisation of the KLPT algorithm to quaternion algebras over a totally real number field.

## How to run the code
Please move to the directory corresponding to the relevant program and follow the instructions of the local `README.md` file.
